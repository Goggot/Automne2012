Arraylist vs Vector

Différences :
	1- Capacité
		-Vector : 	
				--capacité initial
				--gérer le processus d'augmentation de la capacité
				--accès à la capacité
		
		-Arraylist :
				--capacité existe mais impossible d'accéder à une méthode de gestion de capacité
	
	2- Un Vector est synchronisé, et non une Arraylist
	
Les itérateurs 
	- Permet de parcourir les différents éléments d'un vecteur, d'une arraylist, d'un stack, etc.
	
Ex:
	main {
		Arraylist <String> salutations = new Arraylist <String> ();
		salutations.add("Bonjour");
		salutations.add("Salut");
		salutations.add("Good morning");
		etc.
		
		Iterator <String> i = salutations.iterator();	// Doit mettre en oeuvre Iterable
		
		while (i.hasNext()){
			System.out.println(i.next());
			i.remove();
		}
	}
		
	// boucle FOR améliorée
		for(String uneSalutation : salutations)
			System.out.println(uneSalutation);
